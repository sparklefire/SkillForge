## 4. 推荐混合架构

- DGX Spark：原生 Python 工作流、视频转码、抽帧、OCR/Embedding、敏感素材与缓存；本地模型或 VSS 为可选增强。
- Step 3.7 Flash API：多模态理解、SOP规划、证据复核和复杂质检。
- StepAudio ASR/TTS：专家录音转写和培训旁白。
- Step Image Edit：关键帧编辑、海报辅助图，不生成关键操作事实。
- 原始敏感素材默认留在DGX Spark；调用外部API前必须确认赛事与素材授权边界。

培训视频旁白固定使用 `stepaudio-2.5-tts` 和 `zhixingjiejie` 音色。生成器只发送合并后的旁白文本、音色、语音指令和采样率；视频、手册、专家录音、真实面单和Evidence正文不会进入TTS请求。响应音频缓存在Git忽略目录并保持`600`权限，最终成片、生成清单和不含原始媒体的证据包输出到 `output/video/`。
