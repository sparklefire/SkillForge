## 3. 为什么不能直接逐格执行

Notebook明确假设一次性培训环境，并包含五类不能在现有SkillForge部署上原样执行的操作：

- 强制终止3030、8200和11434端口上的进程。
- 删除OpenClaw会话状态和部分ComfyUI输入输出。
- 开启 `dangerouslyDisableDeviceAuth`。
- 自动批准所有待配对设备。
- 把网关令牌打印到Notebook输出和URL片段。

因此复现必须使用独立用户目录、独立状态和回环监听；禁止打印令牌、禁止自动批准设备、禁止清理不属于本次复现的端口或进程。
