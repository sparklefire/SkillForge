# N31 Gold 数据状态

当前没有已确认的 Gold SOP。

在以下条件全部满足前，不得把候选步骤标记为 `VERIFIED`：

1. 连续主视频和关键特写已经完成采集。
2. 专家口述已经录制并校正设备名、时间和参数。
3. 每个必需步骤至少绑定一条可靠证据。
4. “是否执行标签学习”和“成功标准”等现场规则已由操作者确认。
5. 至少一名实际操作者完成逐步审核；最好有第二名复核者。
6. 授权范围覆盖第三方 AI 处理和赛事展示。

审核完成后在本目录保存：

- `gold_sop.json`：符合 `schemas/sop.schema.json`。
- `gold_review.md`：审核结论、争议和允许变体。
- `evaluation_cases.json`：受控遗漏、顺序冲突和无依据内容的正确答案。

受控错误只注入模型草稿或评测副本，不在真实机器上故意制造卡纸或错误介质学习。
