# 汉印 N31 现场采集执行单

目标：采集一条 3-8 分钟、可追溯到手册页码和口述时间点的真实操作视频。不要在机器上故意制造卡纸、装反或错误学习模式；演示错误由系统在 SOP 草稿中注入。

主视频请直接按照同目录下的 [一镜到底拍摄脚本](./一镜到底拍摄脚本.md) 执行。该脚本已经把每一步动作、台词和目标时长固定为5分钟；本执行单用于拍摄前准备和拍摄后验收。

## 二、现场布置

- 手机横屏，1080p、30fps，关闭美颜、滤镜和背景音乐。
- 主机位固定在设备后上方约 45 度，画面同时看到双手、进纸导轨、标签纸入口、走纸键和状态灯。
- 使用均匀照明，避免白色标签纸过曝；先录 10 秒测试画面检查文字和手部动作。
- 环境保持安静，操作者可以边做边说，但另行录制专家口述。
- 装纸和介质展示阶段只展示空白标签。试印使用用户指定的私有历史面单时保持全景，出纸正文不可辨识；打印后先在镜头外检查，再用不透明纸卡遮住全部身份信息、地址、条码、二维码和运单号。
- 用不透明胶带遮挡设备序列号和其他资产编号，不能遮挡操作部件。

## 三、连续主视频

文件名：`cases/n31/input/video/n31_operation_full.mp4`

拍摄时保持一镜到底，失败可以重拍，不要剪掉正常等待过程。下面是候选顺序，是否属于 Gold 必需步骤由操作者和复核者审核。

1. 展示设备整体和型号“N31”，不要拍到序列号。
2. 展示标签纸正面、背面、相邻标签间隙以及宽度测量结果。
3. 说明当前是正常换纸场景；如果机器卡纸，停止主流程，不要强拉。
4. 将标签纸放在能够平直进入机器、不扭曲的位置。
5. 面向机器背面，双手同时调节左右进纸导轨，使其与纸张宽度匹配。
6. 对齐标签纸并平直插入进纸导轨。
7. 拍清机器自动吸纸并从出纸口送出一小段的状态。
8. 说明标签定位类型为缝标，并说明背面照片未发现周期性黑标。
9. 按手册执行一次缝标学习：按住走纸键，在第二次闪灯后立即释放；不要等待第三次闪灯，不执行黑标学习。
10. 短按走纸键，拍清送出一张空白标签。
11. 长按至第一次灯闪烁并释放，打印一张自检页；如不希望消耗标签，可将此项交给复核者决定是否保留。
12. 从本机打印用户指定的私有历史面单；优先复用已成功设置。纸张为 72 × 130 mm，PDF 页面宽 75.85 mm，因此选择“适合可打印区域”，手动缩放时从 94%-95% 开始，不使用 100% 原尺寸。
13. 先在镜头外检查内容完整性，再实体遮挡正文后近距离展示四边、打印边界和停止位置，检查是否居中、无跳纸、无明显偏斜。
14. 口头总结成功标准和发生偏移或跳纸时的下一步处理。

按键依据为用户手册 PDF 第 20 页（印刷页 15）：短按约 0.5 秒走纸；约 1.5 秒至第一次闪灯打印自检页；约 3 秒至第二次闪灯执行缝标学习。本案例已确认是缝标纸，因此第二次闪灯后立即释放，不进入第三次闪灯的黑标学习。现场以状态灯实际反馈为准，不要只按估计时长盲操作。

## 四、必须补拍的五个特写

每段 10-20 秒，文件名固定：

1. `n31_closeup_guides.mp4`：左右导轨随纸宽同步调节。
2. `n31_closeup_media_feed.mp4`：标签纸插入、自动吸入和出纸。
3. `n31_closeup_media_type.mp4`：正反面和缝隙/黑标特征。
4. `n31_closeup_learning.mp4`：手指、走纸键和状态灯同时入镜。
5. `n31_closeup_result.mp4`：自检页，或已实体遮挡正文的历史面单，以及成功/异常判断标准。

## 五、拍摄后验收

- [ ] 主视频为横屏 1080p，时长 3-8 分钟。
- [ ] 调节导轨、插纸、自动吸纸、按键和测试结果都无遮挡。
- [ ] 能从视频判断标签纸的供纸形态和定位方式。
- [ ] 状态灯闪烁和按键释放时机可见。
- [ ] 没有可辨识的真实客户数据、序列号、账号、聊天通知或 API Key；隐私面单不能只依赖后期打码。
- [ ] 音频可听清，没有持续音乐覆盖。
- [ ] 五个特写文件齐全。
- [ ] 没有为了演示故意卡纸、强拉纸张或执行错误模式。

验收通过后，不要压缩或通过聊天软件转发原片；直接复制到项目 `cases/n31/input/video/`。
