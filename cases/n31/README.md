## 用户现在只需做什么

当前没有阻塞自动化开发的用户任务。提交前需要参赛选手完成五项门禁：完整观看80秒成片、180秒计时彩排、最终有声字幕录屏、确认2–5人团队资格、按官方讲义核对提交规则。可选邀请第二名熟悉物流标签打印机的人复核 `gold/gold_review.md`，该项不阻塞当前路演。
