# 汉印 N31 真实主案例

案例编号：`n31_media_change`

案例名称：汉印 N31 电子面单打印机更换标签纸、介质学习与试印验收。

本案例已经替代 SIM 卡方案，作为 SkillForge 唯一真实主案例。目标用户是电商仓库、门店和小型物流团队中的新操作员；核心问题是换纸看似简单，但导轨、介质类型、学习模式和试印验收容易被遗漏，现场经验通常分散在手册、熟练操作员和故障处理习惯中。

## 用户现在只需做什么

1. 拍一张连续两张标签背面的交界处照片；若背面没有周期性黑色定位块，就将介质确认为缝标并采用缝标学习。
2. 按 `materials/一镜到底拍摄脚本.md` 拍摄一条目标5分钟的连续主视频，再按执行单补拍五个关键特写。
3. 按 `materials/专家口述与Gold审核表.md` 录制 2-5 分钟单人语音。
4. 让实际操作者先审核候选步骤，再尽量找第二名熟悉打印机的人复核。
5. 填写 `materials/授权确认书模板.md`，原件放入 `input/authorization/`，不要提交 Git。
6. 通知开发侧素材已经放好，再运行真实摄取、证据对齐和修订评测。

正式拍摄请把纸张自定义为 72 × 130 mm，再打印 `output/pdf/n31_skillforge_test_label.pdf`，选择“实际大小”或“100%”，不要选择适合页面、缩放或自动旋转。历史真实面单和本次含真实物流内容的照片不得出现在演示画面中。

测试面单可重复生成：

```bash
.venv/bin/python scripts/generate_n31_test_label.py
```

拍摄中只使用空白标签或虚构测试数据，不显示真实姓名、地址、手机号、运单号、打印机序列号或后台账号。
