案例编号：`n31_media_change`

案例名称：汉印 N31 电子面单打印机更换标签纸、介质学习与试印验收。

## 用户现在只需做什么

1. 按 `materials/专家口述与Gold审核表.md` 录制 2-5 分钟单人语音。
2. 让实际操作者先审核候选步骤，再尽量找第二名熟悉打印机的人复核。
3. 填写 `materials/授权确认书模板.md`，原件放入 `input/authorization/`，不要提交 Git。
4. 通知开发侧录音和授权已经放好，再开启外部 ASR/视觉理解以及 Gold SOP 固化。

当前实机试印使用私有文件 `cases/n31/input/reference/N31_previous_shipping_label_private.pdf`。优先复用刚才已经成功打印的设置；重新设置时，纸张选择 72 × 130 mm。由于 PDF 页面宽 75.85 mm，大于实物纸宽，不要使用 100% 原尺寸；选择“适合可打印区域”，或从 94%-95% 手动缩放开始并居中。

这份面单只能在本机打印。主视频保持全景，让出纸正文不可辨识；打印完成后先在镜头外检查，拍结果特写前用不透明纸卡遮住姓名、电话、地址、条码、二维码和运单号，只露出纸张四边、打印边界和停止位置。不要只依赖后期打码。打印机序列号、后台账号和通知也不得入镜。
